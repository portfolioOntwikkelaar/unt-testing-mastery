# Unit Testing Mastery (Node.js + Jest)

Dit project is de professional unit testing traject.
Het toont moderne architectuur, Test Data Builders, Humble Object Pattern,
professionele Jest configuratie en een clean codebase.

## Scripts
`npm test` – Run all unit tests  
`npm run test:coverage` – Coverage report

## Topics
- Architectuur voor testbaarheid (SOLID, dependency injection)
- Professionele Jest setup
- Test Data Builders
- Humble Object Pattern
- Unit & integration test structuur
